- 🌟 **Especialização:** JavaScript | React | Vue.js | Next.js | NodeJs | Typescript
- 🎨 **Entusiasta de UX/UI:** Interfaces bonitas, práticas e intuitivas são meu objetivo.
- 🚀 **Apaixonado por inovação:** Sempre explorando novas tecnologias para entregar o melhor.
- 🎯 **Foco no impacto:** Meu trabalho busca unir design e funcionalidade para criar experiências memoráveis.
